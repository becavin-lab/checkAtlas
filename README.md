## Install it from PyPI

```bash
pip install checkatlas
```

## Usage

```py
from checkatlas import BaseClass
from checkatlas import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m checkatlas
#or
$ checkatlas
```
